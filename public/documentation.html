API Documention

HTTP METHOD URL REQUEST Request body data format Response body data format
GET /movies List of all movies A JSON object holding data about all the movies.
Structured like: {
Title: "Shrek"
director: "Andrew Adamson"
genre: "Animation" } 		The full list of movies will show following
the same body format as "request"
GET /movies/:title Data about a movie by title A JSON object holding data about a specific movie.
Structured like:
{ Title: "Shrek"
director: "Andrew Adamson"
genre: "Animation" } 		One movie will show with following info:
Title, Director, Genre
GET /genres/:name Data about a genre by name/title
GET /directors/:name Data about a director by name
POST /users/register Confirmation of successful registration
PUT /users/:id Confirmation of successful user info update
POST /users/:id/favorites Confirmation that a movie has been added to favorites
DELETE /users/:id/favorites/:movieId Confirmation that a movie has been removed from favorites
DELETE /users/:id Confirmation of successful deregistration
