API Documention

HTTP Method URL Request Response body data format
GET /movies List of all movies A JSON object holding data on all the movies.
Structured like: {
Title: "Shrek"
director: "Andrew Adamson"
genre: "Animation" }
GET /movies/:title Data about a movie by title A JSON object holding data on
the specific movie requested.
Structured like: "Results for: " {
Title: "Shrek"
director: "Andrew Adamson"
genre: "Animation" }
GET /genres/:genreName Get a list of movies by genre A JSON object holding data about all the movies
based on the requested genre.
"Results for [genre here]: { title:
director:
genre:
}"
GET /directors/:name Data about a director by name A JSON objecting holding data on a
director with a list of movies by
said director.
"Results for [director name]: { title:
director:
genre:
}"
POST /users/register Confirmation of successful registration A JSON object holding information of
the new user. Structured like: {
name: "Joe Kenda"
email: "Joe_Kenda@email.com"
username:"JoeLikeMovies"
password: "xxxxx5@" }
PUT /users/:id Confirmation of successful user info update A pop up message that the update has been
successful.
"Profile successfully updated!"
POST /users/:id/favorites Confirmation that a movie has been added to favorites A pop up message that the movie was
added to favorites.
"Tittle Here has been added to you favorites!"
DELETE /users/:id/favorites/:movieId Confirmation that a movie has been removed from favorites A pop up message that the movie: "Title Here has
been sucessfully removed".
DELETE /users/:id Confirmation of successful deregistration A pop message confirming cancellation.
"username account has been deleted".
