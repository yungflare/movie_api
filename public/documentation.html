Project 2: My Flix: Documention!

API Documention

Request URL Response
GET /movies List of all movies
GET /movies/:title Data about a movie by title
GET /genres/:name Data about a genre by name/title
GET /directors/:name Data about a director by name
POST /users/register Confirmation of successful registration
PUT /users/:id Confirmation of successful user info update
POST /users/:id/favorites Confirmation that a movie has been added to favorites
DELETE /users/:id/favorites/:movieId Confirmation that a movie has been removed from favorites
DELETE /users/:id Confirmation of successful deregistration
